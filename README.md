# LibraryAPI

# Перед использованием

Для работы LibraryAPI требуется установить .NET SDK 8.0 на машину,
на которой будет работать API.
Установить SDK можно с официального сайта Microsoft
https://dotnet.microsoft.com/en-us/download/dotnet/8.0

# Запуск

Для сборки приложения в .exe файл требуется открыть консоль 
в корневой папке проекта и написать следующую команду:
dotnet build

После сообщения об успешной сборке .exe файл будет находиться 
по директории LibraryAPI/bin/Debug/net8.0/

Так же, перед сборкой рекомендуется установить данные хоста,
адрес и порт, в файле config.json, находящемуся в корневой директории проекта.
(По умолчанию эти значения установлены для запуска на локальный хост)

В качестве базы данных используется файл database.sqlite, что означает,
что для запуска не требуются дополнительные данные базы, так как она хранится локально.

Примечание: файл базы данных создаётся автоматически при первом запуске программы,
изначально он пуст и значения в него можно внести вручную(с помощью редакторов датабаз, 
например, HeidiSQL, или через PUT запрос к API)

# Эндпоинты

# GET /books/
Описание: Получение списка всех книг в библиотеке.

Ответ:

200 OK: Массив объектов книг в формате JSON.

# GET /books/search/{keyword}
Описание: Поиск книг по ключевому слову.

Параметры:

keyword (строка): Ключевое слово для поиска в книгах.
Ответ:

200 OK: Массив объектов книг, соответствующих ключевому слову, в формате JSON.

# GET /books/genre/{genre}
Описание: Получение списка книг, отфильтрованных по определенному жанру.

Параметры:

genre (строка): Название жанра для фильтрации книг.
Ответ:

200 OK: Массив объектов книг, соответствующих указанному жанру, в формате JSON.

# GET /books/{id}
Описание: Получение книги по ее уникальному идентификатору.

Параметры:

id (целое число): Уникальный идентификатор книги для получения.
Ответ:

200 OK: Объект книги с указанным ID в формате JSON.
404 NotFound: Если книга с данным ID не существует.

# PUT /books/
Описание: Добавление новой книги в библиотеку.

Полезная нагрузка: Объект JSON, представляющий добавляемую книгу, со всеми необходимыми полями.

Ответ:

201 Created: Книга успешно добавлена в библиотеку.

# DELETE /books/{id}
Описание: Удаление книги из библиотеки по ее ID.

Параметры:

id (целое число): Уникальный идентификатор книги для удаления.
Ответ:

200 OK: Книга успешно удалена.
404 NotFound: Если книга с данным ID не найдена или не может быть удалена.
